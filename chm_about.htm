歡迎使用 CHM 製作精靈

    「CHM 製作精靈」是一款將網頁文件（Html文檔）轉化為CHM文件（已編譯的 HTML 幫助文件）和將CHM文件轉化為網頁文件的軟件，即網頁「打包」和CHM文件「解包」（CHM文件反編譯）；是集 HTML Help Workshop工程創建，目錄、索引編寫，工程編譯和CHM文件反編譯等多種功能於一身的 CHM電子圖書處理軟件。與HTML Help Workshop v4.74配合使用製作出來的 Html Help 文檔會更見出色。歡迎使用 「CHM 製作精靈」來製作更多 CHM電子圖書！！！

　

軟件開發背景：

1、CHM文件格式已在網上廣為流傳，被稱為一種電子書籍格式。其特點在於，它的每一個幫助頁都是一個Web頁，您可以像瀏覽網站一樣容易地閱讀 HTML 幫助文件。HTML 幫助文件甚至支持 ActiveX, JavaScrip, VBScrip 和 Dll 等。HTML幫助文件類似資源管理器的窗口的瀏覽方式，使用極其方便。另外，還具有管理文檔方便、容量大、壓縮比例高等優點。

2、目前HTML Help Workshop v4.74是製作chm唯一比較流行的工具，但是，存在著效率低，動作簡單重複等缺點；另外，網上也有一些製作chm文檔小軟件，但是，它們多為共享軟件，註冊費少則人民幣十幾元多則上百元，幾乎沒有免費的，至少我沒有找到。

3、製作一種高效工具讓電腦來完成這種簡單重複的工作是必要的，從工程文件的結構來看是可讀的，所以說製作一種高效工具是可行的。
